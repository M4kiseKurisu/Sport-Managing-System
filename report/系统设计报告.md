### 1.3 数据元素表
#### 1.3.1 *用户表*
|  属性名  |    字段名    |  类型   |   约束    |
| :------: | :----------: | :-----: | :-------: |
|   账号   |   account    | varchar | 码1, 唯一 |
|   密码   |   password   | varchar |           |
|    ID    |      id      |   int   | 码2, 主键 |
|   姓名   |     name     | varchar |           |
|   性别   |    gender    | tinyint |           |
| 个人签名 |  signature   | varchar |           |
| 电话号码 | phone_number |   bigint   |    码3    |
|   邮箱   |    email     | varchar | 码4, 唯一 |
#### 1.3.2 *场地表*
| 属性名   | 字段名     | 类型    | 约束      |
| -------- | ---------- | ------- | --------- |
| ID       | id         | int     | 码1, 主键 |
| 位置     | location   | varchar | 码2       |
| 类别     | category   | varchar |           |
| 开始时间 | open_hours | varchar |           |
| 结束时间 |   close_hours     |  varchar    |      |
| 项目限制 |   limit     |  tinyint    |      |
#### 1.3.3 *项目表*
| 属性名           | 字段名      | 类型     | 约束      |
| ---------------- | ----------- | -------- | --------- |
| ID               | id          | int      | 码1, 主键 |
| 名称             | name        | varchar  |           |
| 项目内容（描述） | description | varchar     |           |
#### 1.3.4 *团队表*
| 属性名 | 字段名      | 类型    | 约束      |
| ------ | ----------- | ------- | --------- |
| ID     | id          | int     | 码1, 主键 |
| 名称   | name        | varchar | 码2, 唯一 |
| 描述   | description | varchar    |           |
#### 1.3.5 *器材表*
| 属性名 | 字段名   | 类型    | 约束      |
| ------ | -------- | ------- | --------- |
| ID     | id       | int     | 码1, 主键 |
| 类别   | category | varchar | 码2       |
| 数量   | quantity | int     |           |
#### 1.3.6 *用户运动记录*
| 属性名 | 字段名     | 类型 | 约束      |
| ------ | ---------- | ---- | --------- |
| 用户ID | user_id    | int  | 码1，主键 |
| 项目ID | project_id | int  | 码1，主键 |

| 属性名 | 字段名     | 类型 | 约束      |
| ------ | ---------- | ---- | --------- |
| 团体ID | group_id   | int  | 码1, 主键 |
| 项目ID | project_id | int  | 码1, 主键 |

| 属性名   | 字段名     | 类型    | 约束      |
| -------- | ---------- | ------- | --------- |
| 用户ID   | user_id    | int     |           |
| 项目ID   | project_id | int     | 码1，主键 |
|私有性 | is_private | tinyint |           |
#### 1.3.9 *项目-场地表（场地使用情况）*
| 属性名   | 字段名       | 类型     | 约束      |
| -------- | ------------ | -------- | --------- |
| 项目ID   | project_id   | int      | 码1，主键 |
| 场地ID   | location_id  | int      | 码2       |
| 起始时间 |    start    |  datetime    | 码2  |
| 结束时间 |    end    |   dateime   | 码2  |
#### 1.3.10 *用户-团体表（用户从属团体）*
| 属性名       | 字段名    | 类型    | 约束      |
| ------------ | --------- | ------- | --------- |
| 用户ID       | user_id   | int     | 码1，主键 |
| 团体ID       | group_id  | int     | 码1，主键 |
| 是否为负责人 | is_leader | tinyint |           |
#### 1.3.11 *用户-团体表（用户申请团体）*
| 属性名   | 字段名              | 类型     | 约束      |
| -------- | ------------------- | -------- | --------- |
| 用户ID   | user_id             | int      | 码1，主键 |
| 团体ID   | group_id            | int      | 码1，主键 |
| 申请内容 | application_content | text     |           |
| 申请时间 | application_time    | datetime |           |
| 申请状态 | status       | tinyint  |           |
#### 1.3.12 *用户-用户表（好友交互）*
| 属性名  | 字段名   | 类型 | 约束      |
| ------- | -------- | ---- | --------- |
| 用户ID1 | user_id1 | int  | 码1，主键 |
| 用户ID2 | user_id2 | int  | 码1，主键 |

| 属性名   | 字段名             | 类型     | 约束 |

| 借用数量 | borrowing_quantity | int      |      |
#### 1.3.14 *团体器材借用情况*
